Installation

One command. Done.

Pick your agent. Copy. Paste. Start using 505 expert skills.

/plugins add github:jeffreytse/grimoire

Run inside Claude Code. Installs all 505 skills from the official plugin registry.

curl -sSL https://raw.githubusercontent.com/jeffreytse/grimoire/main/scripts/install.sh | bash

Auto-detects Claude Code, Cursor, Codex, Gemini CLI, and OpenCode. Installs to all found agents.

./scripts/install.sh --domain engineering

Install one domain only. Replace engineering with any of the 26 available domains.

./scripts/install.sh --skill engineering/development/propose-commit

Install a single skill by path. Use the <domain>/<subdomain>/<skill-name> pattern.

Quality Standard

What makes a grimoire skill

Every skill passes 4 gates before merge. No vague advice. No generic tips.

✅

Actionable

Reader can do it immediately. Steps, not descriptions. Commands, not concepts.

🏛️

Proven

Adopted by top-tier institutions — Google, McKinsey, WHO, ABA, NSCA. Cited, auditable, third-party verified.

🎯

Specific

Names tools. Cites numbers. Shows exact commands. One skill, one concept. No overlap.

🤝

Consensus

Encodes majority position among experts. No 50/50 debates. Stable, durable knowledge.
