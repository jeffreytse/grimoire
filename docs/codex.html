Install Grimoire for Codex

grimoire --target codex

FAQ

Codex + Grimoire

Where does Grimoire install skills for Codex?

Skills are installed to ~/.agents/skills/. Codex CLI reads AGENTS.md and skills from this directory automatically on startup.

Does Grimoire auto-detect Codex?

Yes. The grimoire install script checks for the codex binary and installs to ~/.agents/skills/ automatically when Codex is detected.

Can I install only engineering skills for Codex?

Yes. Run grimoire --domain engineering --target codex to install only the engineering domain.

Does Grimoire work with all Codex versions?

Yes. Grimoire uses the AGENTS.md standard which Codex CLI supports from v0.1.0 onwards.
