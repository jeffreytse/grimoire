FAQ

Copilot + Grimoire

How do I install Grimoire for Copilot?

Run copilot plugin install grimoire@grimoire from your terminal. Copilot CLI will download and register the Grimoire plugin automatically.

Does the grimoire shell script install to Copilot?

Yes. Running the grimoire script with --target all installs to all detected agents including Copilot. Auto-detection checks for the copilot binary.

Can I use Grimoire skills in Copilot Chat?

Grimoire skills work with Copilot CLI. For Copilot Chat in VS Code, install the grimoire extension or use the shell script method.

Do I need a GitHub Copilot subscription?

Yes. You need an active GitHub Copilot subscription. Grimoire itself is free and open-source — it only requires Copilot to be installed and authenticated.
