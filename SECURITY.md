# Security Policy

## Reporting a Vulnerability

**Do not open a public issue for security vulnerabilities.**

Report via [GitHub Security Advisories](https://github.com/jeffreytse/grimoire/security/advisories/new)
or email the maintainer at the address on their [GitHub profile](https://github.com/jeffreytse).

Include a description, steps to reproduce, affected files, and potential impact.

## Response Timeline

| Stage | Target |
| ----- | ------ |
| Acknowledgement | 48 hours |
| Fix or mitigation | 30 days (critical), 90 days (others) |

## Supported Versions

Only the latest release receives security fixes.
