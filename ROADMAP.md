# Roadmap

grimoire has 32 skills. It needs 500. Every domain below is open for contribution.

See [CONTRIBUTING.md](./CONTRIBUTING.md) and [STANDARD.md](./STANDARD.md) before writing a skill.
Use [SKILL_TEMPLATE.md](./SKILL_TEMPLATE.md) as your starting point.
Run `scripts/validate-skill.sh your/SKILL.md` before opening a PR.

Labels: `good first skill` = beginner-friendly, `advanced` = requires deep domain expertise.

## How to contribute

1. Pick a wanted skill above
2. Run: `cp SKILL_TEMPLATE.md skills/<domain>/<subdomain>/skills/<skill-name>/SKILL.md`
3. Fill in the template — every field is required
4. Run: `scripts/validate-skill.sh skills/<domain>/<subdomain>/skills/<skill-name>/SKILL.md`
5. Open a PR — one skill per PR preferred

Questions? Open an issue with the `skill-request` template.
